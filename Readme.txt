Detalles
----------------------
Título: 	              LevelRange
Intefaz:                11200
Versión:                2.0.3

Autores:                Bull3t, Tenyar97, rado-boy, blehz.
Página Web del Addon:   https://github.com/Tenyar97/LevelRange-Turtle

Funcionalidades actuales:
 [*] Muestra los rangos de nivel de las zonas, diplomacia, bandas e instancias en el Mapa del Mundo.
 
Lista de comandos:
 [*] /lr help >> Mostrar lista de comandos.
 [*] Mostrar el cuadro de opciones de LevelRange:
  [*] /lr
  [*] /levelrange
 [*] /lr toggle >> Mostrar u ocultar el tooltip de LevelRange.
 [*] /lr instances >> Mostrar u ocultar las instancias en el tooltip.
 [*] /lr pvp >> Mostrar u ocultar la diplomacia PvP en el tooltip.
 [*] /lr fishing >> Mostrar u ocultar el nivel mínimo de pesca en el tooltip.
 [*] Recargar la interfaz de usuario:
  [*] /rui
  [*] /reloadui

Recomendaciones:
 [*] Si las zonas de tu facción aparecen como "Hostil" recarga la interfaz utilizando uno de los dos comandos mencionados anteriormente.
 [*] Si el nivel de pesca no aparece en el tooltip, desactiva la opción de mostrar nivel de pesca requerido y vuelve a activarla.
